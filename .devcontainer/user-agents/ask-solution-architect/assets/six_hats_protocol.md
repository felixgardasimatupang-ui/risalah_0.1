# Six Thinking Hats Protocol

## The 6 Personas

- **White Hat (Facts)**: Extract data, identify gaps, formulate data queries.
- **Red Hat (Emotions)**: Identify stakeholder & UX friction points.
- **Black Hat (Caution)**: Identify single points of failure, severe bottlenecks, catastrophic risks.
- **Yellow Hat (Benefits)**: Highlight maximum upside, systemic benefits, value proposition.
- **Green Hat (Creativity)**: Formulate lateral alternatives, technological workarounds.
- **Blue Hat (Process)**: Synthesize outputs into an actionable summary.

## Execution Rules
- MUST analyze the problem using one Hat entirely before the next.
- Prefix steps with the active Hat inside the `<process>`.
